Boo! Nice to meet you.
I am here to scare all your problems away by keeping track of your tasks.
What can I help you with today?

Supported Commands:
1. list -> Provides a list of existing tasks.
2. mark X -> Marks task number X as done.
3. unmark X -> Marks task number X as undone.
4. todo taskName -> Creates a todo task with name taskName.
5. deadline taskName /by date -> Creates a deadline task with name taskName and deadline date.
6. event taskName /from startDate /to endDate -> Creates an event task with name taskName,
   start date startDate, and end date endDate.
7. delete X -> Deletes task number X from the list.
8. on givenDate -> Displays all the tasks that occur on givenDate.
9. help -> Prints the list of commands supported by this bot.

Please enter dates in the format of either yyyy-MM-dd hh:mm or yyyy-MM-dd.

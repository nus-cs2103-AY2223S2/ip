## Getting Started

To begin using Saturday, download the latest release [here](INSERT LINK TO DOWNLOAD). 

Once you've run the application, you will be greeted with a clean and intuitive dashboard. On the right pane will be your task list and on the left will be a chat box where you can talk to Saturday and give it commands.

## Commands

Saturday has a lot of cool commands to help you get things organized and have fun at the same time. Here are just a few:

### Create a ToDo task
---
A **ToDo** has **no timeframe**.
<p align="center">
  <img width="640" src="/assets/todo.png">
</p>

```
todo {task description}
```

### Create a Deadline task
---
A **Deadline** has **one deadline timeframe**.
<p align="center">
  <img width="640" src="/assets/deadline.png">
</p>

```
deadline {task description} /by {hh:mm dd MM yyyy}
```

### Create an Event task
---
An **Event** has **one start timeframe** and **one end timeframe**.
<p align="center">
  <img width="640" src="/assets/event.png">
</p>

```
event {task description} /from {hh:mm dd MM yyyy} /to {hh:mm dd MM yyyy}
```

### Mark a task
---
**Mark** a task as **done**.
<p align="center">
  <img width="640" src="/assets/mark.png">
</p>

```
mark {index #}
```

### Unmark a task
---
**Mark** a task as **undone**.
<p align="center">
  <img width="640" src="/assets/unmark.png">
</p>

```
unmark {index #}
```

### Delete a task
---
**Delete** a task.
<p align="center">
  <img width="640" src="/assets/delete.png">
</p>

```
delete {index #}
```

### List
---
**View** your **entire task list**.
<p align="center">
  <img width="640" src="/assets/list.png">
</p>

```
list
```

### List by date
---
**View** tasks on a **specific date**.
<p align="center">
  <img width="640" src="/assets/list-on.png">
</p>

```
list /on {hh:mm dd MM yyyy}
```

### Find a keyword
---
**Find** tasks with a specific **keyword**.
<p align="center">
  <img width="640" src="/assets/find.png">
</p>

```
find {keyword}
```

### Sort by time
---
**Sort** your task list by **time**.
<p align="center">
  <img width="640" src="/assets/sort.png">
</p>

```
sort
```

### Sort by index
---
**Sort** your task list by **index**.
<p align="center">
  <img width="640" src="/assets/unsort.png">
</p>

```
unsort
```

### Exit
---
Stop Saturday.
```
bye
```


## FAQ

__Q:__ How do I transfer my data to another Computer?
__A:__ Install Saturday and overwrite the empty data file with the data of Saturday on your original machine.
